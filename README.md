# フロンティアAI（ミュトス）脅威対策 対応ダッシュボード

金融機関向けに、フロンティアAI（ミュトス）脅威への **短期的緊急対応策** の整備状況を
「見える化」するデモ用ダッシュボードです。経営層〜現場が同じスコアカードで状況を共有できます。

## 内容
- 9つの短期緊急対応策を **5段階** で評価（スコアカード）
- 対応成熟度 **レーダーチャート** / 項目別 **対応段階バー**
- **対応策の5ステップ**（Consult / Modernize / Protect / Recover / Operate）
- **期限 × 進捗トラッカー**（期限・主担当・ステータス：完了／対応中／未着手）
- デモ用サンプル3社（A社／B社／C社）の切替・全社比較

## 使い方
`index.html` をブラウザで開くだけで動作します（グラフ描画に Chart.js CDN を利用）。

## 構成
- `index.html` … 画面・スタイル
- `app.js` … データ・描画ロジック

## カスタマイズ
`app.js` の `COMPANIES`（社名・スコア）、`ITEMS`（対応策・期限・主担当）を編集してください。

> ※スコア・社名はデモ用サンプル、期限／主担当は元Excelの値です。
